4. Por que a utilização de Java em multiplataforma é muito importante para os programadores? 
R = pois possibilita que um mesmo programa seja executado em diversas plataformas.
Um programa escrito na linguagem Java é compilado e gera um arquivo de bytecodes (com extensão .class), que pode ser executado onde quer que a plataforma Java esteja presente, em qualquer sistema operacional subjacente. 
Em outras palavras, o mesmo programa pode ser executado em qualquer sistema operacional que execute a plataforma Java.
